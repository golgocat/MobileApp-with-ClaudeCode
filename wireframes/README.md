# Wireframes Directory

Add your UI mockups and design screenshots here.

## How to Use

1. **Take screenshots** of your design mockups
2. **Paste or save** them in this directory
3. **Reference them** when asking Claude Code to implement UI

## Tips for Better Results

- Include both light and dark mode designs
- Show different screen sizes if needed
- Include interaction states (pressed, disabled, etc.)
- Add annotations for specific requirements

## Example Structure

```
wireframes/
├── home-screen.png
├── login-screen.png
├── profile-screen.png
├── components/
│   ├── button-variants.png
│   └── card-designs.png
└── flows/
    ├── onboarding-flow.png
    └── checkout-flow.png
```

## In Claude Code

Reference wireframes by saying:
> "Implement the design from wireframes/home-screen.png"

Or paste screenshots directly into the Claude Code chat.
